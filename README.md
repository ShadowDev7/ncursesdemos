# Ncurses-Demo

Εδώ βάζουμε σύντομα προγράμματα που δείχνουν πως προγραμματίζουμε με τη βιβλιοθήκη ncursesw, έκδοση 5.

Απαιτεί να έχουμε εγκαταστήσει το πακέτο ```libncursesw5-dev```, μπορείτε να χρησιμοποιήσετε αυτήν την εντολή στο Debian/Ubuntu:

```bash
sudo apt-get install libncursesw5-dev
```

Με ```make``` γίνεται η μεταγλώττιση των προγραμμάτων.

Με make ```clean``` καθαρίζουμε τα μεταγλωττισμένα.

Για να βάλετε και εσείς σύντομα προγράμματα, [δείτε το πακέτο Ncurses](https://directory.fsf.org/wiki/Ncurses)

Το newdemo προέρχεται από τον υποκατάλογο test/ του παραπάνω πακέτου.

-------------------------------------------------------------------------------------------------------------------------------

Here we add short programs that demonstrate how do we program with the library ncrusesw, version 5

You are required to install the package ```libncursesw5-dev```, you can use this command on Debian/Ubuntu:

```bash
sudo apt-get install libncursesw5-dev
```

With ```make`` you compile the programs

With ```make clean``` you clean the compiled 

To add your own short programs, [check out the Ncurses library](https://directory.fsf.org/wiki/Ncurses)

Newdemo comes from the sub-list test/ of the above package.
